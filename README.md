# Wallet Guard Bot

## Описание
Telegram-бот, работающий на GitHub Actions, который отслеживает регулярные счета и присылает уведомления о необходимости оплаты.

## Установка и настройка
1. Создайте бота в @BotFather.
2. Скопируйте `config.json.example` в `config.json` и внесите свои счета.
3. Добавьте секреты `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в Actions.
4. Запустите Workflow вручную для проверки.

## Расписание
Запуск каждый день в 10:00 UTC. Бот присылает напоминание, если до платежа осталось 2 дня или меньше.
